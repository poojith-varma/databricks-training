# 📘 Week 2 – Day 3 Assignment: Understanding Core SQL Functions (Date, NULL, Number, String)

This assignment is designed to improve understanding of essential SQL function categories through organized exercises and practical data transformation scenarios.

---

# 📚 Topics Covered (SQL Practice – Essential Function Concepts)

---

## 🔹 1. Working with Date Functions

- Retrieving date components (YEAR, MONTH, DAY, DAYNAME)
- Finding differences between two dates
- Applying date formatting and conversions
- Using functions related to current date and time
- Filtering records using date ranges and conditions

---

## 🔹 2. NULL Value Handling & Data Preparation

- Using IS NULL / IS NOT NULL conditions
- Replacing NULL values with IFNULL and COALESCE
- Applying NULLIF for conditional NULL handling
- Managing missing salary, bonus, and transaction information
- Cleaning and preparing incomplete datasets for analysis

---

## 🔹 3. Numeric Function Operations

- Performing ROUND, CEIL, FLOOR, and TRUNCATE operations
- Using MOD for remainder calculations
- Applying POWER and SQRT for mathematical processing
- Using ABS for absolute number conversion
- Formatting numeric values using FORMAT
- Comparing values with GREATEST and LEAST

---

## 🔹 4. String Function Operations

- Changing text case using UPPER and LOWER
- Extracting text with SUBSTRING, LEFT, and RIGHT
- Joining strings using CONCAT
- Manipulating strings through pattern-based functions
- Performing basic text formatting and cleanup

---

## 🔹 5. Applying SQL in Business Scenarios

- Combining multiple functions for practical use cases
- Performing salary-related calculations and modifications
- Implementing discount and payment calculation logic
- Transforming data using layered SQL functions
- Managing incomplete or unavailable data efficiently

---

## 🔹 6. Building Queries with Multiple Concepts

- Combining STRING, NUMBER, and NULL functions together
- Using nested function transformations within SELECT statements
- Creating derived columns for analytical purposes
- Preparing transformed data for reports and dashboards

---
