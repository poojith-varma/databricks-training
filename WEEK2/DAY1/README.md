## 📚 Concepts Covered Today
🔹 **Database Design**  
- Schema creation for CMS, CRS, FDS  
- Primary keys & foreign keys  
- Relationships between tables  

🔹 **Data Insertion & Testing**  
- Sample rows for each schema  
- Ensuring queries return meaningful results  

🔹 **SQL Queries**  
- Basic `SELECT` statements  
- Joins (`INNER`, `LEFT JOIN`)  
- Aggregations: `SUM()`, `AVG()`, `COUNT()`  
- Grouping with `GROUP BY`  
- Filtering with `HAVING`  
- Sorting with `ORDER BY`  

🔹 **Advanced SQL Concepts**  
- Distinct counts (`COUNT(DISTINCT ...)`)  
- Window functions (`SUM() OVER`, `COUNT() OVER`, `AVG() OVER`)  
- Combining aggregates with row-level detail
