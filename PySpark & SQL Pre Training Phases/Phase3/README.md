## Phase 3

## Objective

Build a simple ETL pipeline using PySpark to read customer and sales data, clean invalid records, perform data transformations, generate business insights, and create a final reporting table.

## Topics Covered

* Reading CSV and JSON files
* Data validation and cleaning
* Filtering records
* Joining DataFrames
* GroupBy and aggregations
* SQL and PySpark DataFrame operations
* Building an ETL pipeline
* Reporting and data analysis
