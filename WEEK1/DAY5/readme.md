# Day 5 (Advanced SQL Window Functions )

## Summary

In Day 5, I deepened my understanding of SQL window functions by exploring advanced analytical techniques. I practiced calculating moving averages, grouping data using percentile-based ranking with `NTILE()`, and identifying the first and last transactions for each customer using `ROW_NUMBER()`.
I also learned how to combine CTEs with aggregation to compute structured results like total sales per employee. Additionally, I practiced real-world analytics such as department payroll calculations and employee contribution percentages.


## 🛠 Tools Used
- MySQL
- DB-Fiddle
- GitHub


## Concepts Covered Today

- AVG() OVER() with ROWS BETWEEN (moving averages)
- NTILE() for bucket-based segmentation
- First and last record per group using ROW_NUMBER()
- Department-wise aggregations using PARTITION BY
- Filtering based on window function results
- Payroll and contribution percentage calculations
- COUNT(*) OVER() for total row-level analytics
- CTE (Common Table Expressions) for structured query breakdown
- Total sales per employee using aggregation + CTE
- Combining GROUP BY with window functions
